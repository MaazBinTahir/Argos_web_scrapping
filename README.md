# Argos_web_scrapping
Scrapping the Bedding catalog from Argos' Website

Using Python to scrape bedding catalog from Argos website.
A total of 258 products were scrapped along with 31 features including:

1) Name of Product
2) Url of product
3) Brand
4) Sizes
5) Price
6) Description
7) Dimensions
8) Reviews
9) Rating
10) Recommendations
11) ... and more

## Libraries used:

1) Requests
2) Beautiful Soup
3) Pandas
